# Percolation

Following type of lattice site percolation simulation.

 - Square lattice
  - nearest neighbor
  - next nearest neighbor
 - Triangular lattice
  - nearest neighbor
 - d dimensional simple lattice
  - nearest neighbor

To learn how to use this package, see example/percolation.ipynb.
